```

┌─────────────────────▼───────────────────────────────────┐
│                   Nginx（リバースプロキシ）                 │

└──────┬──────────────────────────────────────────────────┘
       │                           │
┌──────▼──────┐             ┌──────▼──────┐
│  React SPA  │             │  FastAPI    │

                            ┌──────▼──────┐

                                   │
                            ┌──────▼──────┐
                            │ データローダー │

                            └──────┬──────┘

                            ┌──────▼──────┐
                            │ 財務CSVデータ │
                            │ (経理部提供) │
                            └─────────────┘
```

## コンポーネント説明

### フロントエンド（React + TypeScript）
- Single Page Application
- ルーティング：React Router v6
- 状態管理：React Query（APIデータキャッシュ）
- UIコンポーネント：Tailwind CSS
- チャート：Recharts（標準）+ D3.js（カスタム）

### バックエンド（FastAPI）
- KPI計算済みデータのREST API提供
- 認証：Azure AD OAuth2（JWTトークン検証）
- データ取得：SQLAlchemy ORM
- レスポンスキャッシュ：Redis（将来対応）

### データベース（PostgreSQL）
- 財務データ・KPI計算結果の永続化
- 時系列クエリ最適化（インデックス設計）

### データローダー（Python + pandas）
- 経理部提供CSVをDBに投入
- KPI計算ロジック（`経理部/指標計算ロジック/kpi_calculator.py`）を呼び出し
- 決算発表後の定期バッチ実行

## 開発環境構築手順
```bash
# リポジトリクローン後
cd AI_dashboard_dev/ソフト開発部/インフラ
docker compose up -d

# フロントエンド開発サーバー
cd src/frontend && npm install && npm run dev

# バックエンド開発サーバー
cd src/backend && pip install -r requirements.txt && uvicorn main:app --reload
```
