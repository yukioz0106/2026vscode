# KPI計算値 検証レポート

## 概要
有価証券報告書（2021〜2025年3月期）から抽出した財務データを用いて、
KPI一覧（経営企画部定義）の各指標を算出し、開示値との照合を実施した。

## 照合結果：ROE（F-01）

| FY | 算出値 | 開示値 | 差異 | 判定 |
|----|-------|-------|------|------|
| FY21 | 4.9% | 4.9% | 0.0pt | ✓ 一致 |
| FY22 | 20.7% | 20.7% | 0.0pt | ✓ 一致 |
| FY23 | 5.0% | 5.0% | 0.0pt | ✓ 一致 |
| FY24 | 9.5% | 9.5% | 0.0pt | ✓ 一致 |
| FY25 | 7.1% | 7.1% | 0.0pt | ✓ 一致 |

> 開示値は有価証券報告書「主要な経営指標等の推移」p.2の「親会社所有者帰属持分当期利益率」と照合。

## 重要な計算上の注意事項

### ROE計算方式の確定
- **採用方式**: 期中平均自己資本 = (前期末自己資本 ＋ 当期末自己資本) / 2
- **誤った方式（期末値）では差異が生じる**（FY22: 18.8% vs 開示20.7%）
- `kpi_calculator.py`の `FinancialData.equity_prior` フィールドに前期末自己資本を必ずセットすること

### FY25セグメント変更に関する注意
- JX金属（金属事業）がFY25にIPOして非継続事業に分類された
- FY25のP/L・セグメントデータは「継続事業」ベースの値
- FY24の比較値も同様に組み替え済み（有報注記参照）

### EBITDA計算の留意点
- CFの「減価償却費及び償却費」はリース除却・非継続事業含む全体値
- セグメント表の減価償却費はリース使用権資産を含む（継続事業合計）
- FY24: CF記載値334,054百万円 / セグメント計上値294,632百万円（差は金属事業分）
