# 経理部

## ミッション
有価証券報告書・決算短信等の開示資料から財務データを抽出・構造化し、
経営企画部が定義したKPIを正確に算出できるデータ基盤と計算ロジックを整備する。

## 主要タスク

### 1. 財務データの整備（→ `財務データ/`）
- 連結P/L・B/S・CF計算書の過去5期分をCSV化
- セグメント別損益データの整備
- 四半期データの時系列整合チェック

### 2. 指標計算ロジックの確定（→ `指標計算ロジック/`）
- KPI一覧（経営企画部定義）の各算出式をコード化
- 計算結果の検証（開示値との突合）
- 異常値・欠損値の処理ルール策定

### 3. マスタデータの管理（→ `マスタデータ/`）
- 勘定科目マスタ（開示科目名→内部名称のマッピング）
- 期間マスタ（FY・四半期の定義）
- セグメントマスタ

## 主な成果物ファイル
| ファイル | 内容 |
|---------|------|
| `財務データ/consolidated_pl.csv` | 連結損益計算書（5期分） |
| `財務データ/consolidated_bs.csv` | 連結貸借対照表（5期分） |
| `財務データ/consolidated_cf.csv` | 連結CF計算書（5期分） |
| `財務データ/segment_data.csv` | セグメント別損益・資産 |
| `指標計算ロジック/kpi_calculator.py` | KPI算出Pythonスクリプト |
| `マスタデータ/account_master.csv` | 勘定科目マスタ |

## 財務データ抽出元
| 資料 | 用途 |
|------|------|
| 有価証券報告書（2021〜2025年3月期） | 年次確定値・注記情報 |
| 決算短信（FY20〜FY24） | 速報値・四半期データ |
| 決算説明会資料 | セグメント詳細・計画値 |

## 連携先
- ← 経営企画部：KPI算出に必要なデータ項目リストの受領
- → ソフト開発部：整備済みCSV・計算ロジックの提供
- → 品質評価部：計算結果の検証依頼
