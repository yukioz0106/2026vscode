# 経営ダッシュボード開発プロジェクト

## プロジェクト概要
ENEOSグループの財務・非財務データを統合し、経営層がリアルタイムで意思決定できる
世界最高水準の経営ダッシュボードを構築する。

## 参照データ
- ソース: `../AI_dashboard/ENEOS/`
  - 有価証券報告書（2021〜2025年3月期）
  - 統合レポート（2021〜2025年）
  - 決算短信・説明資料（FY20〜FY24）
  - 中期経営計画（第3次）

## 部門構成

| 部門 | 役割 | 主な成果物 |
|------|------|-----------|
| [経営企画部](./経営企画部/) | 戦略整合・KPI設計 | 中期経営計画読解、KPI体系、要件定義 |
| [経理部](./経理部/) | 財務データ整備・指標算出 | 財務マスタ、計算ロジック、決算指標 |
| [ソフト開発部](./ソフト開発部/) | ダッシュボード実装 | 設計書、ソースコード、インフラ構成 |
| [品質評価部](./品質評価部/) | 品質保証・UX評価 | テスト計画、評価レポート、改善提案 |

## 開発フロー
```
経営企画部（KPI・要件定義）
    ↓
経理部（データ整備・指標計算ロジック確定）
    ↓
ソフト開発部（実装）
    ↓
品質評価部（テスト・評価）→ フィードバック → 経営企画部
```

## ターゲット指標
- ROE / ROIC / ROA
- EBITDA / 純利益 / フリーキャッシュフロー
- 有利子負債比率（D/Eレシオ）
- セグメント別営業利益
- 中計KPI達成度（非財務含む）
