# 経営企画部

## ミッション
中期経営計画を読解・構造化し、ダッシュボードに映すべき経営指標・KPI体系を定義する。
経営層・各部門の情報ニーズを要件に落とし込み、開発の羅針盤を作る。

## 主要タスク

### 1. 中期経営計画の読解・構造化（→ `中期経営計画/`）
- ENEOSグループ第3次中期経営計画の財務目標・非財務目標を抽出
- 戦略マップ（BSC形式）の作成
- 計画値 vs 実績値の対比表の設計

### 2. KPI体系の定義（→ `KPI定義/`）
- 財務KPI（ROE・ROIC・EBITDA等）の定義と算出式の確定
- 非財務KPI（CO2排出量・人的資本等）の定義
- KPIオーナー部門のアサイン

### 3. 要件定義（→ `要件定義/`）
- 経営層へのヒアリング結果の整理
- ダッシュボード画面一覧・機能要件
- 非機能要件（レスポンス・セキュリティ・可用性）

## 主な成果物ファイル
| ファイル | 内容 |
|---------|------|
| `中期経営計画/中計サマリー.md` | 第3次中計の財務・非財務目標の構造化 |
| `KPI定義/KPI一覧.md` | 全KPIの定義・算出式・データソース |
| `要件定義/画面要件定義.md` | ダッシュボード画面・機能の要件 |

## 連携先
- → 経理部：KPI算出に必要なデータ項目をリスト化して依頼
- → ソフト開発部：画面要件定義・ワイヤーフレームを提供
- → 品質評価部：KPI定義を評価基準として共有
