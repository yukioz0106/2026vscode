## P-01 経営サマリー画面 詳細

### レイアウト
```
┌─────────────────────────────────────────────┐
│  ENEOS経営ダッシュボード    2026-Q1          │
├──────────┬──────────┬──────────┬────────────┤
│  当期純利益 │   ROIC   │   FCF    │  D/Eレシオ │
│  ¥2,800億 │   7.8%   │ ¥3,200億 │   0.52倍  │
│  計画比▲6% │ 目標▲0.2pt│ 計画比+7%│  目標▲0.02│
├──────────┴──────────┴──────────┴────────────┤
│  [売上収益・営業利益 四半期推移グラフ]       │
├─────────────────────┬───────────────────────┤
│  セグメント別営業利益  │  中計KPI達成メーター   │
│  （積み上げ棒グラフ）  │  （ゲージチャート）    │
└─────────────────────┴───────────────────────┘
```

### 機能要件
- KPIカード：実績値・計画値・前年比・計画比を表示
- カラーコード：計画達成=緑、±5%=黄、未達=赤
- 期間フィルタ：四半期・年度・中計3ヵ年を切替可能
- ドリルダウン：各KPIカードクリックで詳細画面へ遷移

## 非機能要件

| 区分 | 要件 |
|------|------|
| レスポンス | 初期表示3秒以内、フィルタ切替1秒以内 |
| セキュリティ | 認証あり（社内SSO）、未公開データのアクセス制御 |
| 可用性 | 稼働率99%以上（経営会議前後は特に重要） |
| ブラウザ対応 | Chrome最新版・Edge最新版 |
| 対応デバイス | PC（1920×1080基準）、タブレット対応 |
| データ更新 | 決算発表後24時間以内に反映 |
