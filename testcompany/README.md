# ホームページ作成会社 — AI運営体制

## 会社概要
Claude Codeによって運営されるAIエージェント型ホームページ制作会社。
各部門に世界レベルの専門AIエージェントを配置し、受注から納品まで自動化する。

## 組織図

```
testcompany/
└── 社長/                  ← CEOエージェント（意思決定・統括）
    ├── 経営企画/          ← 戦略・マーケ・事業計画
    ├── 経理/              ← 財務・請求・税務
    ├── 制作/              ← フロントエンド・デザイン・実装
    └── 品質管理/          ← テスト・検査・納品承認
```

## 各エージェントの役割

| 部門 | エージェントファイル | 主な責務 |
|------|---------------------|---------|
| 社長 | 社長/agent.md | 全体統括・最終意思決定 |
| 経営企画 | 社長/経営企画/agent.md | 戦略・営業・市場分析 |
| 経理 | 社長/経理/agent.md | 財務・請求・税務 |
| 制作 | 社長/制作/agent.md | ウェブ制作・デザイン |
| 品質管理 | 社長/品質管理/agent.md | 品質保証・納品検査 |

## 案件処理フロー

```
顧客からの依頼
    ↓
社長：案件受注判断
    ↓
経営企画：提案書・見積もり内容策定
    ↓
経理：見積書発行・入金確認
    ↓
制作：ホームページ制作
    ↓
品質管理：検査・承認
    ↓
社長：最終納品承認
    ↓
経理：請求書発行
```

## エージェントの使い方

各 `agent.md` をClaude Codeのシステムプロンプトとして読み込むことで、
その部門の専門家として振る舞うエージェントを起動できる。

```bash
# 例：社長エージェントとして起動
claude --system-prompt 社長/agent.md

# 例：制作エージェントとして起動
claude --system-prompt 社長/制作/agent.md
```
